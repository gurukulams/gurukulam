Handling Multiple-Strings

What is Multiple-Strings..?

Multiple-Strings

  • In previous chapter we learn about C string is a one dimensional character array.
  • Then what is an array of C string or Multiple-Strings looks like?
  • It's a two dimensional character array!

Declaration and Initialization of Multiple-Strings

  • The General Format of Declaration and Initialization of Multiple-Strings or an array of Strings are shown below:-
    •           
              char arr_name[NUMBER_OF_STRING][MAX_STRING_SIZE] =
              {
                "Strings",
              };
  • Here, the NUMBER_OF_STRING represents that how many strings are inserted and MAX_STRING_SIZE represents the maximum size of the string.
  • For-Example:-
    •           
              char ch_arr[3][10] = {
                  {'s', 't', 'u', 'd', 'e', 'n', 't', '\0'},
                  {'c', 'l', 'a', 's', 's', '\0'},
                  {'t', 'e', 's', 't', '\0'}
              };
  • The Another Method of Declaration and Initialization is:-
    •           
              char ch_arr[3][10] = {
                  {"student"},
                  {"class"},
                  {"test"}
              };

Example Program

Example Program for Multiple-Strings 

      
              #include<stdio.h>
              int main()
              {
                  int i;
              
                  char student[4][10] = {
                      "Gokul",
                      "Oviya",
                      "Kanishk",
                      "Illakiyan"};
              
                  char City[4][10] = {
                      "Trichy",
                      "Coimbatore",
                      "Chennai",
                      "Coimbatore"};
              
                  printf("Name & City of The Students: \n\n");
              
                  for (i = 0; i < 4; i++)
                  {
                      printf("Name : %s \t City: %s \n", student[i], City[i]);
                  }
                  return 0;
              }
