Introduction

What is ‘C’?

  • General-purpose, Procedural, Imperative Computer Programming Language.
  • Developed at AT & T Bell Laboratory, USA, 1972.
  • Designed and written by ‘Dennis Ritchie’.

Why 'C'?

  • Structured and powerful programming language
  • Windows, LINUX, RDBMS, Android are written in C.
  • First choice for IOT Devices

Getting Started

  • Learn alphabets, digits and special symbols.
  • Combine them to form constants, variables and keywords.
  • Form an instruction.
  • A group of instruction form a program.

Constants, Variables and Keywords

  • Constant => entity that doesn’t change.
  • Otherwise called Literals.
  • Variable => entity that may change.
  • Otherwise called Identifiers.
  • Keyword => special meaning word.

Types of Constants

Primary Constants

  • Integer Constatnt
  • Real Constant
  • Character Constant

Secondary Constants

  • Array
  • Pointer
  • Structure
  • Union
  • Enum

Rules for constructing Integer Constants

  • at least one digit.
  • no decimal point.
  • Either positive or negative.
  • Default sign is positive.
  • No commas or blanks are allowed within an integer constant.
  • The allowable range for integer constants is -32768 to 32767.

Rules for constructing Real Constants

  • At least one digit.
  • Have a decimal point.
  • Either positive or negative.
  • Default sign is positive.
  • No commas or blanks.

Rules for constructing Character Constants

  • Must be a single alphabet, a single digit or a single special symbol enclosed within single inverted commas.

Rules for constructing Variable Names

  • Combination of 1 to 31 alphabets, digits or underscores.
  • First character must be an alphabet or underscore.
  • No commas or blanks .

First C Program


            #include
            int main() {
            /* my first program in C */
               printf("Hello, World! \n");
              return 0;
            }
  • #include => preprocessor command, compiler to include stdio.h file.
  • int main() => main function => program execution begins.
  • /*...*/ => additional comments and ignored by compiler.
  • printf(...) => message "Hello,World!" displayed on the screen.
  • return 0; terminates the main() function and returns the value 0.
