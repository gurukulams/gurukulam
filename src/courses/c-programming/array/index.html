Introduction to Array

Array-Definition

  • An Array is the collections of data-items of same datatype.
  • The array is a type of data structure that is used to store homogeneous data in contiguous memory locations.
  • It is a collection of data of same type stored in consecutive memory cells under one name.
  • Each individual element in the array can be referenced by indexing. Index are always start at 0.

Declaration of Array

Array-Declaration

  • The General Format of the Array-Declaration is:-
  • typeName ArrayName[size] = { list of values }
  • This declares an array with the specified size, named ArrayName and type typeName.
  • The array is indexed from 0 to (size-1).
  • The size (in brackets) must be an integer literal or a constant variable.
  • The compiler uses the size to determine how much space to allocate (i.e. how many bytes).
  • Assume that, if we declare the array as., int a[5]. Then it looks like below:-
  • Here, the array a is declared with the size 5. so the index value is start from 0 & size-1.

Initialization of an Array

Array-Initialization

  • After an array is declared it must be initialized. Otherwise, it will contain garbage value(any random value).
  • An array can be initialized by two ways.
  • They are:-
    • Compile-Time Initialization
    • Run-Time Initialization
  • Compile-Time Initialization:-
    • Compile time initialization of array elements is same as ordinary variable initialization.
    • The General Format is:-
    • data-type array-name[size]={list of values};
    • Examples:-
    • int marks[5]={ 73, 37, 10, 33, 77 };
  • Run-Time Initialization:-
    • An array can also be initialized at runtime using scanf() function.
    • This approach is usually used for initializing large arrays, or to initialize arrays with user specified values.
    • Examples:-
                          
                  int arr[4];
                  for(i = 0; i < 4; i++)
                  {
                  scanf("%d", &arr[i]); //Run time array initialization
                  }

Advantages of Array

  • Random access of elements using array index.
  • Easy access to all the elements.

Disadvantages of Array

  • Allows a fixed number of elements to be entered which is decided at the time of declaration. Unlike a linked list, an array in C is not dynamic.
  • Insertion and deletion of elements can be costly since the elements are needed to be managed in accordance with the new memory allocation.

    • Accessing Array Elements

    How to Access an Array Elements..?

    • Array elements are accessed by using an integer index.
    • Array index starts with 0 and goes till size of array minus 1.

    Passing Array Elements to a Function

    Passing array element by element to function :

    • Individual element is passed to function using Pass By Value parameter passing scheme.
    • Original Array elements remains same as Actual Element is never Passed to Function. Thus function body cannot modify Original Value.
    • Suppose we have declared an array ‘arr[5]’ then its individual elements are arr[0],arr[1]…arr[4]. Thus we need 5 function calls to pass complete array to a function.
