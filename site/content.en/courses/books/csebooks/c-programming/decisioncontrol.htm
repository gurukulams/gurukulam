Decision Control Instructions

  • In C programs, the collection of instructions and these instructions are executed one after another from top to button sequentially.
  • But some times we have to take decision while doing programming like executing some portion of the program instead of the whole program so we use decision statements .
  • Decision Control statements we can able to take decision on basis of one condition.
  • If condition is evaluates to true (1) then we will execute some particular block and if it is false (0) then we will execute another block.

Types of Decision Control Instructions

  • In C we have three different types of decision control statements are there.
    • if Statements
    • if-else Statements
    • Nested if Statements

if-else

if Statement

  • Syntax:-
    •     
        if (condition)
        { 
            Statements 1; 
            Statements 2; 
            Statements 3; 
        }
  • Syntax:-
    •     
        if (condition)
        { 
            Statement1; 
            Statement2; 
        }
        else
        { 
            Statement3; 
            Statement4; 
        }

Nested if-else

  • Syntax:-
    •     
        if(condition1)
        { 
            Statements 1; 
            Statements 2; 
            Statements 3;  
        }
        else_if(condition2)
        { 
            Statement 4;
            Statement 5; 
        }
        else 
        Statement 6;
