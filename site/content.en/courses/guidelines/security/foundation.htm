Security - Guidelines

Security Principles


  • Least Privilege Principle
  • Defenders Have to Protect Everything, Attackers Need Just One Weak Spot
  • Simplicity Leads to Stronger Security

Process

  • Secure by design
  • Zero trust
  • Fail Fast (Release often)

Code security

  • Upgrade by default
  • Zero deprecation
  • Automation first
  • Back-end should not be aware of front-end

Data security

  • Hashing > Encryption
  • Denided persistence

API Security

  • JWT Token
    • opaque token
    • Custom registration flow
  • OPEN ID Integration
